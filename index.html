Featured Project

'마음토닥'

MBTI & 별자리 기반 심리상담 챗봇

1. 문제 정의

현대인들은 일상 속 다양한 스트레스와 감정적 어려움을 겪지만, 이를 편하게 털어놓을 창구가 부족합니다. 전문적인 상담은 비용과 시간의 장벽이 있고, 지인에게 이야기하기에는 부담을 느낍니다. '마음토닥'은 언제 어디서든 익명으로 감정을 털어놓고 위로를 받을 수 있는 AI 챗봇 서비스입니다.

2. 아키텍처 / 설계

사용자에게 즉각적이고 안정적인 서비스를 제공하기 위해 경량 웹 프레임워크와 로컬 LLM을 결합한 아키텍처를 설계했습니다.

  • 백엔드: Python Flask를 사용하여 RESTful API 서버를 구축, 프롬프트 동적 생성 및 로컬 LLM 통신 담당.
  • 프론트엔드: Vanilla JavaScript, HTML, CSS만으로 동적인 SPA 형태의 채팅 UI를 구현하여 가볍고 빠른 UX 제공.
  • AI 모델: Ollama를 통해 로컬 환경에서 LG EXAONE 모델을 구동. (외부 API 의존성/비용 문제 해결, 프라이버시 확보)

3. 핵심 기능

페르소나 기반 감성 대화 생성

사용자의 기분, MBTI, 별자리 정보로 AI의 페르소나('마음토닥')와 역할, 말투, 답변 규칙을 상세하게 정의한 프롬프트 엔지니어링을 적용했습니다. Flask 서버에서 프롬프트를 동적으로 완성해 로컬 EXAONE 모델에 전달, 단순 정보 제공을 넘어 감정에 공감하는 개인화된 위로 메시지를 생성합니다.

비동기 통신을 활용한 동적 채팅 UI/UX

Vanilla JavaScript의 fetch API를 사용하여 백엔드와 비동기 통신하는 채팅 인터페이스를 구현했습니다. 메시지 전송 시 새로고침 없이 즉시 표시되며, '입력 중...' 애니메이션과 타이핑 효과를 제공하여 실제 대화와 같은 몰입감과 UX를 향상시켰습니다.

4. 나의 역할 및 기여

  • Flask 기반 백엔드 서버 아키텍처 설계 및 API 엔드포인트(generate) 구현
  • AI 페르소나 정의 및 동적 프롬프트 생성 로직 설계 등 프롬프트 엔지니어링 주도
  • Ollama를 활용하여 로컬 LLM(EXAONE)을 연동하고 응답을 처리하는 로직 구현
  • Vanilla JS를 사용한 비동기 채팅 UI 개발 및 사용자 경험(타이핑 효과 등) 개선

5. 결과 및 성과

  • 외부 API 없이 로컬 LLM을 성공적으로 연동하여 월 고정 비용 0원의 서버 운영 환경 구축
  • 정교한 프롬프트 엔지니어링을 통해 사용자의 감정에 깊이 공감하는 인간적인 위로 메시지 생성 기능 구현
  • 경량 기술 스택(Flask, Vanilla JS)을 사용하여 초기 로딩이 빠르고 반응성이 뛰어난 웹 애플리케이션 완성

6. 회고 (배운 점 및 개선점)

배운 점
  • Flask 백엔드 구축과 프론트엔드 비동기 통신 과정의 깊이 있는 이해
  • 프롬프트 엔지니어링의 중요성과 AI 페르소나 제어 노하우 습득
개선점
  • 대화 기록(History)을 관리하여 더 깊이 있는 상담 기능 고도화
  • 스트리밍(Streaming) 응답 방식으로 사용자 경험 향상

7. 동작 흐름

  1. 사용자 입력: 사용자가 웹페이지에서 MBTI, 별자리, 기분을 입력하고 '전송' 버튼을 누릅니다.
  2. 지시문 생성: 서버는 전달받은 사용자 정보와 미리 정의된 AI의 역할(페르소나, 말투, 규칙)을 조합하여 AI를 위한 상세한 지시문을 만듭니다.
  3. AI 답변 생성: 이 지시문을 로컬 exaone 모델에 전달하면, 모델이 지시문에 맞춰 따뜻한 위로의 메시지를 생성합니다.
  4. 화면에 표시: 생성된 메시지가 다시 사용자 웹페이지로 돌아와 채팅창에 나타납니다.
